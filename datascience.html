Real Estate Price Prediction Model

Leveraged Machine Learning and Python

Introduction

Gaining insights into the factors that influence property values can significantly impact decision-making processes. This project will provide a guide on how to construct an efficient house price prediction using Python and Machine Learning.

Data Collection

Web scraping techniques were used to collect information from www.realtor.com.

Data Cleaning

Throughout the data cleaning process, I maintained consistent and transparent communication with my supervisor, ensuring a collaborative and open exchange of information.

To ensure the consistency and reliability of the collected data, data cleaning was performed. Null values were removed and dropped. A new “zip” column was created.

Using statistics, extreme outliers were removed by checking the price per sqft and removing those beyond one standard deviation.

Created a function to remove outliers:

Permutation Feature ranks feature importance based on the impact each feature has on the trained machine learning model’s predictions. The results show that the sqft feature has the highest importance with the number of baths following as the second most important.

Building Machine Learning Model

LinearRegression was used because inputs such as number of beds and baths, zip code, and sqft have a linear relationship to the target variable, price.

Evaluating Estimator Performance

K-fold Cross-Validation

Performance measure is reported by dividing the dataset into K equally sized subsets and then training and evaluating K times. The final performance is determined by averaging the performance.

Grid Search CV

Different algorithm models are executed with various parameters to obtain the optimal model.

Results show that LinearRegression model with parameter “normalized=False” is best.

GradientBoostingRegressor

Attempts to boost accuracy are then utilized.

Boosting the accuracy score using GradientBoostingRegressor did not indicate a higher accuracy score.

Conclusion

In summary, LinearRegression demonstrates the highest accuracy score among the models considered, establishing it as the most reliable model.

Since the model was created on a test dataset, it must be considered with real-time data. By comparing the price with Zillow in 2023, the difference in price was 4.34%.
